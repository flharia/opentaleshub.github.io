# Open Tales Hub

A free storytelling website with dark theme for comfortable night reading.

## Features
- Dark theme optimized for night reading
- Mobile responsive design
- Adjustable font sizes
- No ads, completely free

## Visit
https://opentaleshub.github.io

## How to Add Stories
1. Edit `index.html`
2. Add new story cards in the "stories" section
3. Commit changes
